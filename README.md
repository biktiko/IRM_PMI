# Վաճառքի կետերի գնահատման դաշբորդ (Streamlit)

**Լեզու՝ ամբողջ UI-ն հայերեն է։**

Ֆայլի կառուցվածք
- Բազաներ՝ թերթեր, որոնց անվանման մեջ կա `BR` (`BR1`, `BR2`, `BR3`, `BR4`, `YAP-BR 5` և այլն)։
- Քաշեր՝ `LAS_Brand Retail(1..4)` և `YAP-Brand Retail(5)` (կամ մեկ ընդհանուր `Weights`)։
  *C*՝ հարցի տեքստը, *E*՝ հարցի քաշը բաժնի ներսում, *F*՝ հարցի քաշը սցենարի մակարդակով։

Գնահատման տրամաբանություն
- «Այո/Yes/Да/Ուն/Есть» ⇒ 1, «Ոչ/No/Нет/Չունի» ⇒ 0, մնացածը՝ NA (չի ազդում):
- Հարցի սքոր բաժնի մակարդակով՝ E-ով կշռված, սցենարի մակարդակով՝ F-ով կշռված։
- Խանութի սքոր՝ սցենարների միջին տոկոսը (ֆիլտրերով կառավարելի):
- Ընթացիկ գնահատական (1–10)՝ առանձին ցուցանիշ՝ միջին ըստ խանութների/սցենարների։

Քայլեր
```
pip install -r requirements.txt
streamlit run app.py
```
